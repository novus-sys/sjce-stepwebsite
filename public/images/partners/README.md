# Partner Logos

Please add the following logo files to this directory:

## Required Logo Files:
1. `cii.png` - CII (Confederation of Indian Industry)
2. `startup-india.png` - Startup India
3. `iit-madras.png` - IIT Madras
4. `startupindia-hash.png` - #startupindia
5. `jss-step.png` - JSS STEP
6. `govt-karnataka.png` - Government of Karnataka
7. `jss.png` - JSS
8. `tie-mysuru.png` - TiE Mysuru
9. `msme.png` - MSME
10. `aim.png` - AIM (Atal Innovation Mission)
11. `startup-karnataka.png` - Startup Karnataka
12. `nasscom.png` - NASSCOM
13. `make-in-india.png` - Make in India
14. `nif.png` - National Innovation Foundation

## Logo Requirements:
- Format: PNG with transparent background preferred
- Size: Minimum 200x200px, maximum 500x500px
- Aspect ratio: Square or close to square works best
- File size: Keep under 100KB each for optimal loading

## How to Add Logos:
1. Save each partner's logo with the exact filename listed above
2. Place them in this `/public/images/partners/` directory
3. The component will automatically display them

Note: If a logo file is missing, the component will show a broken image. Consider adding fallback placeholder images if needed.
