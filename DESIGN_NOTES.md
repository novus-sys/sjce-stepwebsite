# SJCE-STEP Website - Design System

## Color Palette

### Primary Colors
- **Navy Blue**: `#000080` - Main SJCE-STEP brand color (backgrounds, primary elements)
- **Orange**: `#ff6b35` - Accent color for highlights, CTAs, and icons
- **White**: `#ffffff` - Backgrounds, text on dark backgrounds

### Secondary Colors
- **Light Navy**: `#1a1a99` - Hover states
- **Light Orange**: `#ff8c5a` - Secondary accents

### Neutral Colors
- **Gray-900**: `#1a1a1a` - Body text
- **Gray-600**: `#666666` - Muted text
- **Gray-100**: `#e5e5e5` - Borders

## Design Principles

1. **Navy, Orange, White Combination**: Inspired by modern incubator aesthetics
2. **Orange for Energy**: Used for CTAs, icons, and highlighted text
3. **Navy for Trust**: Professional backgrounds and primary elements
4. **White for Clarity**: Clean backgrounds and contrast
5. **Focused Content**: 3 startups per row (1 row on home page)

## Component Styling

### Buttons
- Primary: Navy blue background, white text
- Outline: Navy border, navy text, hover fills with navy

### Cards
- White background
- Gray-200 border
- Hover: Navy border (30% opacity)
- Icons: Navy blue in 10% opacity background

### Badges
- Navy blue 10% background
- Navy blue text
- No borders

### Typography
- Headlines: Navy blue for emphasis
- Body: Gray-900 for readability
- Links: Navy blue

## Sections

1. **Hero**: Clean layout with navy accents, professional stats
2. **Why Choose Us**: 6 features with consistent navy icons
3. **Featured Startups**: 3 startups only, navy theme
4. **Testimonials**: Professional cards with navy accents
5. **Insights**: Blog preview with navy badges
6. **CTA**: Solid navy background

## Best Practices

- Use `text-primary` instead of gradients
- Use `bg-primary/10` for icon backgrounds
- Keep borders subtle with `border-gray-200`
- Hover states use `hover:border-primary/30`
- All icons use `text-primary` color
